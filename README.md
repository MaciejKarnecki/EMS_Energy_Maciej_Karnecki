# EMS Energy

Projekt Energy Management System wykonany w Ignition 8.3 z wykorzystaniem bazy danych PostgreSQL.

Autor: Maciej Karnecki  
Grupa: 31-INF-PSI-SP-A  
Moduł: Energy

## Zawartość repozytorium

- `sql/` – skrypt SQL tworzący bazę danych EMS Energy
- `ignition/` – backup projektu Ignition
- `dokumentacja/` – dokumentacja techniczna PDF
- `screenshots/` – zrzuty ekranu z projektu

## Funkcje systemu

- monitorowanie liczników energii elektrycznej
- zapis danych pomiarowych do PostgreSQL
- archiwizacja historii tagów
- dashboard KPI
- widok liczników
- widok szczegółowy licznika
- widok lokalizacji
- widok analityczny z filtrowaniem
- alarmy i zdarzenia
- pipeline alarmowy
- role i strefy bezpieczeństwa
